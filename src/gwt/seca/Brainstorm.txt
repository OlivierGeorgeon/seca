*Feedback présentation du 16/06/2011 :
	Olivier Voisin // Projet IDEAL
	
	Olivier est chargé de faire une version 3D interactive de l'environnement de simulation de ERNEST 
	agent artificiel devant exhiber un comportement d'apprentissage dévelepemmental.
	
	L'objectif est de permettre aux utlisateurs d'Ernest (qui est disponible sur le web) d'interagir avec l'environnement 
	-> modifier l'environnement d'ERNEST http://e-ernest.blogspot.com/
	
	Le travail consiste donc à étudier les différents environnements existants, de mettre en évidence les propriétés recherchées 
	pour l'usage dans le cadre d'une recherche fondée sur la validation par simulateurs des architectures et algorithmes associés 
	à l'apprentissage dévelopemental.
	
	La contribtion : SECA (Simulation Environment for Constructivist Agents) est interactif, immersif, en ligne, et multi-agent. 
	Le codage est de type web et le code peut évoluer en ligne (contenu de la page) -> évolutif, interactionaliste -> matrices visuelles 
	et tactiles comme stimuli, Analyse on line des comportements (trace based...).
	
	Le code exploite la carte graphique directement -> performances, mais... moins de généricité naturellement
	
	Zoom sur la réalisation
	
	- maths -> celles des étagères de la recherche et du web
	- architecture avec des "moteurs" (simulation, rendu)
	
	Questions : quelle est la problématique de recherche (Nathalie).
	Réponse : c'est une question de recherche en ingénierie des simulations comportementales (pour l'apprentissage dévelopemental). 
			(vision d'Alain...) -> aller jusqu'au bout de la logique de recherche -> aller vers une forge ?
